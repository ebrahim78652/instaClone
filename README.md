# instaClone
A simple insta clone made with ReactJS, Express, MongoDB. 

# Features: 
1) Follow User
2) Unfollow user
3) Sign in
4) Sign up
5) Authentication done by JWT
6) Make a Post
7) Make a Comment
8) Search for a User
9) Like and Unlike a Post

10) Yet to implement: editing comment, editing description of post, deleting post.
